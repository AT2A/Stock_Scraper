# Stock News Scraper

## Description
 The scraper scrapes google finance, stockanalysis.com, and yahoo finance to retrieve the days stock information,
 its all displayed in a txt file and gemini ai is used to summarize all the information which is also included in the txt file

### There's a simple GUI using tKinter that takes the stock ticker and stock exchange of the stock
### That information is used to get the link to the webage on google finance and Stock Anaylsis to scrape the financial and analyst data, it is also used to get multiple news articles from the yFinance API which the content of each of the articles is also scrapped
### The information is put into a text file and is sent to the Gemini API and is summarized and the summary is dislplayed in the text file.
